Projects

IMDB Movie and Tv Analysis

This Rust project analyzes a dataset of top movies and TV shows from IMDB, building a graph connecting movies and directors based on metascores. It then calculates the degree centrality of each node.

Pixel Runner Game

Pixel Runner Game is a simple 2D runner built with Python's Pygame library. Players control a character jumping over moving snails to score points.

Flight Data Analysis

This project aims to provide comprehensive exploratory data analysis (EDA) of flight data, offering valuable insights into patterns, trends, and anomalies.

Editor AI

Editor AI is a collaborative project under Spark that aims to create an application to assist micro newspapers with editing through an AI-powered style editor.

MAPLE (Bill Summarization, Tagging, Explanation)

This project simplifies Massachusetts bills for the MAPLE Platform by generating summaries and category tags using machine learning. It aims to make legal language more understandable (9th-grade level) and includes a pipeline for bill processing, a dashboard for text display and saving, and integration into the MAPLE platform.The code for the project is on my github however, the arrow link belows goes to the deployed project.

Data Driven COVID-19 Policy Recommendations

In our Data Science 310 project, we utilized Power BI and SQL to analyze COVID-19 data from specific regions worldwide. Our goal was to identify the most effective practices for reducing infection rates.

Money Moves Game

In the Money Moves project from the Fintech Focus Program in the JPMorgan cohort, to developed a web application aimed at improving financial literacy. Using Python, Flask, MongoDB, HTML, and CSS, we created a quiz presented in a game format.

Sentiment Bender

SentimentBender is a project that dives into the complex emotional states of key characters in the renowned animated series "Avatar: The Last Airbender." Using natural language processing techniques, AI, sentiment analysis tools, and data visualization, the series is decoded and analyzed to discover the sentiments expressed by characters throughout the series.
